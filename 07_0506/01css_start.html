css소개와 문법

css는 cascading style sheets 의 약자로서 html문서의 내용에 서식을 적용하기 위한 언어로서 미국의 개발자 '에릭 메이어'라는 개발자에 의해서 개발된 언어이다.

현재까지 css level3까지 나온 상태로서 웹문서의 레이아웃, 그림자효과, 애니메이션 등의 동적인 콘텐츠도 개발이 가능한 상태이다.

css의 특징은 웹문서 제작 시 다양한 사용자 환경에서 동일한 내용을 볼 수 있게 표현해주는 언어로서 웹표준 제작 시 필수 언어이다.

1.css 문법(작성법)

css언어는 기본적으로 head와 head 사이에 작성한다. 문법은 '선택자{속성:값;}'으로 작성한다.

1) 선택자(selector) - 서식을 적용하기 위한 하나의 방법을 말하며 대략 50가지가 있다.

2) 속성(property) - 속성은 서식의 종류로서 예를 들면 글자모양, 글자크기, 줄간격, 배경색, 가로크기, 세로크기 등의 서식내용을 말한다.

3) 값(value) - 속성을 선택하면 반드시 설정하는 내용을 말한다. 예를 들면 'font-size:20px;background:#f00'

2. css의 특징

css언어는 모든 문서에 동일한 서식을 만들 수 있으며, 어느 환경에서나 사용자가 의도한 디자인을 구현할 수 있다. 유지보수 측면에서도 편리한 장점이 있다.

css언어는 html태그처럼 주석을 표시할 수 있는데 '/* 주석내용 */'으로 작성한다.

3. css 작성 시 위치 지정 방법

1) 내부스타일 - 현재문서에서만 스타일 서식을 적용하는 방법

2) 외부스타일 - 별도의 css문서를 작성 저장하여 html문서에 연결 적용하는 방법

3) 인라인스타일 - 적용하고자 하는 태그요소안에 직접 스타일 서식을 작성하여 적용하는 방법으로서 주로 테스트용도로 사용된다.
