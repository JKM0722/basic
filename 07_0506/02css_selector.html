02css_selector(선택자) - id, class, tag

서식을 적용하기 위한 방법인 기본 선택자를 사용하여 문서서식 적용하기

1. class(클래스)선택자 - .class명

클래스 선택자는 사용자가 지정한 영역에 서식을 적용하기 위한 선택자로서 한번 작성된 선택자는 언제든 다시 사용할 수 있는 서식을 적용할 때 사용한다. clss선택자는 사용자가 이름을 작성하여 적용하는데, 이름을 작성할 때 의미있는 이름으로 작성하며 이름앞에 '.'기호를 붙여주는 것이 특징이다.

2. id(아이디)선택자 - #id명

아이디 선택자는 클래스 선택자와 마찬가지로 사용자가 지정한 영역에 서식을 적용하기 위한 선택자로서 클래스 선택자와 다른 점은 서식을 반복해서 사용하지 않고, 중복되지 않는 서식을 만들고자 사용한다. 아이디 이름 작성 시 이름앞에 '#'기호를 붙인다.

3.tag(태그)선택자 - tag명

사용자가 지정한 태그를 문서 안에서 찾아 서식을 적용하기 위한 선택자이다. 서식이 한꺼번에 변경되기 때문에 실제 사용 시 원하지 않는 부분까지도 적용되기 때문에 주의해야한다.
